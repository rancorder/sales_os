# DRIVE_REFERENCES｜代表Drive参照一覧

このファイルは全社の「代表Drive参照」だけを管理する。
個別資料（商談動画、文字起こし、SU定例、SCデータ、スライド等）のIDは、各 `clients/*.md` の6章「追加資料ID一覧」に追記する。

## 運用ルール
- 代表Drive参照は、案件の入口となる file / folder を1件だけ置く。
- Drive IDが増えた場合、原則として各社カルテ6章へ追記する。
- 代表参照を差し替える場合のみ、このファイルと該当カルテ6.1を同時更新する。
- GitHubには原本ファイルを置かず、Drive ID / URL の参照に留める。
- URL列は `https://drive.google.com/file/d/{ID}/view` または `https://drive.google.com/drive/folders/{ID}` として開く。
