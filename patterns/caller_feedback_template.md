# 架電者FBテンプレ

## 目的
架電者が勝手に台本を変えたり、意図と違う話し方になった時に、責めずに再現性を戻す。

## 基本文
```md
@{架電者名}さん

実際の架電でお願いしたいのは、まず修正せずに台本通り対応することです。

今回の台本は、単に文章として作っているのではなく、
受付突破→本人接続→課題切り分け→アポ化のどこで落ちるかを検証するためのものです。

現場で違和感がある場合は、自己判断で変更するのではなく、
ログ・録音・相手の反応をもとに共有いただければ、こちらで台本側を修正します。

次回からは、まず現行台本通りで運用をお願いします。
そのうえで、反応が悪い箇所を一緒に改善していきましょう。
```

## 台本遵守チェック項目
- 冒頭1文を変えていないか
- 商材説明を早く出していないか
- 禁止ワードを使っていないか
- 必須質問を聞けているか
- 断り時に次回接点を取っているか
- アポ判定メモが残っているか

## KPI
- 台本遵守率
- 必須ヒアリング回収率
- 禁止ワード発生率
- 次回接点獲得率
