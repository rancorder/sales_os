# GPTプロジェクトチャット抽出プロンプト

このチャットを削除する前提で、営業OSに移すべき情報だけを抽出してください。

以下の3分類で出力してください。

## 1. clients/{社名}.md に追記する情報
- 商材の核
- ターゲットの歪み
- 商談/架電の勝ち筋
- 進捗ログ
- 次の一手
- 資料原本のfileIdやURL

## 2. patterns/ に保存する横断知見
- 他案件にも使える型
- 架電/商談/FB/報告テンプレ
- NG訴求
- KPI分析観点
- 勝ちパターン

## 3. docs/ に保存する営業OS運用知見
- GitHub/Drive/Codex/Claude/GPT/Vercel運用
- 自動化構想
- フォルダ設計
- エージェント起動プロンプト

## 出力ルール
- Markdownで出力
- 雑談、感情整理、一時的な言い換え、却下済み案は除外
- 事実と推測を分ける
- 未確認は `{未確認}` と書く
- 個人情報、電話番号、メールアドレス、顧客リスト原本は出さない
- 保存先パスを明記する
