## 2. 商材の核 / USP

### 2.1 商材A：電子基板材料

Drive内の営業資料上、株式会社センドはパナソニックインダストリー製の高品質な半導体パッケージ基板を中心に取り扱う電子材料の専門商社として整理されている。

核は以下。

```txt
日本製・小ロット・短納期で、日本のものづくりの調達課題を支える。
```

主な取り扱い：

```txt
・半導体パッケージ基板
・Sem-3 / セムスリー
・FR4
・プリント基板の土台となる材料
```

強み：

```txt
1. パナソニックインダストリー製の品質
2. パナソニックとの直接取引による価格・供給面の優位性
3. 国内在庫を活かした小ロット・短納期対応
4. 1枚から相談可能な試作フェーズ適性
5. 提携加工会社との連携による加工済み基板の納品
6. 顧客からの質問・トラブル相談をセンドが一括対応
```

### 2.2 商材Aで刺さる課題

刺さるのは「高品質な基板があります」ではなく、以下の調達・開発スピードの詰まり。

```txt
・海外製材料では納期が2〜3ヶ月かかり、急な試作要求に間に合わない
・完成品メーカーから短納期試作を求められる
・商社経由だと高く、対応が画一的
・小ロットで材料を取りたいが、既存ルートでは動きにくい
・材料選定、加工先、納期確認、相談窓口が分散して面倒
```

営業上の変換：

```txt
材料売りではなく、試作・小ロット・短納期の調達保険として売る。
```

## 3. 営業設計

### 3.1 最重要方針

この案件の最重要論点は、商材A/Bを混同しないこと。

```txt
電子基板材料：調達・試作・短納期・小ロット・品質保証の営業
湘南ベースレインボー：子育て社員支援・福利厚生・家庭学習支援の営業
```

同じ「センド」案件でも、受付突破先、担当部署、刺さる課題、商談で確認する論点がまったく違う。

### 3.2 商材A：電子基板材料のターゲット

優先ターゲット：

```txt
S：プリント基板加工メーカー
理由：完成品メーカーから急な試作・短納期依頼を受けやすく、材料調達の詰まりが発生しやすい。

A：電子機器メーカー / IoT機器メーカー / 次世代家電 / 電動モビリティ / 新興メーカー
理由：開発スピード・小ロット試作・柔軟な調達ルートが価値になりやすい。

B：商社経由の材料調達に不満があるメーカー
理由：価格・対応スピード・相談窓口の一本化を訴求できる。
```

想定接続先：

```txt
・調達担当
・購買担当
・開発責任者
・基板設計/製造部門
・試作/量産立ち上げ担当
・代表/事業責任者（ベンチャー、新興メーカー）
```

### 3.3 商材A：受付突破の入口

強い入口：

```txt
急な試作案件や小ロット材料調達で、海外材の納期が合わないケースが出た際に、国内在庫・短納期で相談できる基板材料の件です。
```

避ける入口：

```txt
高品質な半導体パッケージ基板のご案内です。
```

理由：品質訴求だけだと「既存ルートで足りている」で止まりやすい。最初に出すべきは、調達遅延・試作遅延・小ロット不可の痛み。

### 3.4 商材A：商談で確認する論点

```txt
・現在扱っている基板材料の種類
・Sem-3 / FR4 の利用有無
・海外材/国内材の比率
・急な試作・小ロット依頼の頻度
・調達で詰まりやすい納期・価格・品質のどれが課題か
・既存商社/メーカー直接取引で困っている点
・材料だけでよいのか、加工済み納品まで必要か
・見積時に必要な仕様情報
・次回は調達/開発/技術の誰を巻き込むべきか
```

### 3.5 商材B：湘南ベースレインボーのターゲット
