## 2. 商材の核 / USP

KANSHAは、単なる動画制作会社ではなく、企業イベント・展示会・セミナー・記者発表・採用活動で発生する言葉、熱量、説明、反応を、営業・採用・広報で再利用できる「映像資産」に変える会社として扱う。

### 2.1 中核訴求

```txt
企業イベントを、営業で使える映像資産に変える。
```

展示会やセミナーを、その日限りで終わらせず、営業・採用・広報活動で中長期的に機能し続ける高付加価値資産へ変換する。

### 2.2 強み

- 企画・構成・演出から撮影・編集・仕上げまで一貫対応。
- 地上波テレビ、CM、企業VP、映画等で培った制作ノウハウ。
- ドローン、360度VR、複数カメラLIVE配信、字幕・資料合成に対応。
- 舞台監督・進行ディレクター経験者を含む現場対応力。
- Webマーケティング、SNS拡散、AR、イベント制作、グラフィックなど周辺領域も対応可能。
- 著作権譲渡・二次利用・用途別再編集を商談上の武器にできる。

## 3. 商談設計

### 3.1 基本方針

サービス説明から入らない。まず相手のイベント・展示会・採用活動・既存動画の使われ方を棚卸しする。

悪い入口：

```txt
動画制作できます。
```

良い入口：

```txt
展示会やセミナーで、来場者数や名刺は残る一方で、社長の言葉、技術者の説明、来場者の反応、会場の熱量が、その日限りで消えていないか。
```

### 3.2 初回商談で確認する論点

- 直近で展示会・セミナー・記者発表・採用イベント・社内イベントがあるか。
- 既存動画は、Web掲載だけで終わっていないか。
- 営業・採用・広報・IR・社内研修で二次利用できているか。
- 社長・役員・技術者・顧客の言葉を残せているか。
- 納品後の再編集・権利・素材管理に課題があるか。
- 直近イベントを、営業素材として回収する余地があるか。

### 3.3 業態別分岐

| ターゲット | 刺すべき論点 | 商談で確認すること |
|---|---|---|
| 中堅・大企業 | 社内調整、ブランド管理、部署横断利用、採用・営業・広報への多展開 | 主管部署、既存パートナー、法務・ブランド確認、二次利用範囲 |
| 広告代理店 | 制作パートナー、短納期、撮影/編集のみ、クレジット・NDA対応 | 立ち位置、依頼パターン、窓口ルール、権利処理 |
| イベント会社 | 回線・音声・冗長化、複数カメラ、ハイブリッド配信、事故防止 | 開催形態、内製範囲、機材、直近トラブル、バックアップ体制 |
| 採用課題企業 | 入社後ギャップ防止、職場理解、説明の均質化 | 採用対象、既存採用動画、説明会、現場社員の出演可否 |
| 製造・技術系企業 | 技術者説明、展示会ブース、代理店教育、営業ショート化 | 直近展示会、営業資料、技術説明者、販売店・代理店活用 |

### 3.4 二次商談化条件

次回化は「詳しい提案をします」では弱い。以下を満たした場合に二次商談へ進める。

- 直近イベント・展示会・採用活動・記者発表など素材収穫日がある。
- 既存動画の二次利用・再編集・用途展開が弱い。
- 営業部・採用部・広報部など複数部署で使える余地がある。
- 相手が「動画の目的」「使う場面」「社内共有先」を言語化できていない。
- 用途に近い事例を提示すれば検討が進む。
