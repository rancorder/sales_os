根本思想：

```txt
日本人の肌には、日本由来のものが合う。
肌につけるものも、食べ物と同じように、その土地に根付いた植物・薬草・伝承成分が合う。
```

## 3. 営業設計

### 3.1 重要な方針修正

前回のWeb調査時点では、人事・採用責任者向けの研修・福利厚生仮説を置いていた。  
しかし、キックオフ議事録上の営業ターゲットは、旅館、エステ、美容室、ネイルサロン、美容皮膚科、クリニック、介護、訪問介護、ケアマネージャー周辺、インバウンド向け店舗である。

したがって、現時点の営業OSでは以下を正とする。

```txt
主戦略：福美水を、業態ごとの顧客体験を上げる装置として提案する。
入口：商品説明ではなく、相手業態ごとの導入理由から入る。
初回導線：サンプル送付 → 使用感確認 → オンライン商談/訪問商談。
```

人事・採用責任者向け仮説は、現時点では主戦略から外す。  
もし既存トークスクリプトに人事・採用呼び出しが残っている場合は、実際の主商材・ターゲットとズレている可能性があるため再修正対象。

### 3.2 業態別の置く理由

| 業態 | 置く理由 | 初回訴求 |
|---|---|---|
| 高級旅館 | 高級アメニティ、宿泊体験、インバウンド土産、館内物販 | 「宿泊後に購入につながる日本由来スキンケア」 |
| エステ | 敏感肌客に提案しやすいホームケア | 「施術後も自宅で続けられる低刺激ケア」 |
| 美容皮膚科・クリニック | 肌トラブル層への低刺激ケア | 「治療後/敏感肌の患者様にすすめやすいケア」 |
| 介護・訪問介護 | 高齢者の乾燥ケア、入浴後保湿、家族提案 | 「高齢者の乾燥ケアをやさしく支える商品」 |
| インバウンド店舗 | 日本由来、抹茶、富士山、土産需要 | 「日本らしさが伝わるスキンケア土産」 |
| 美容室・ネイル | 既存顧客への物販・ケア提案 | 「施術外売上につながる低刺激スキンケア」 |
| 薬局・調剤薬局 | 肌が弱い人への信頼導線 | 「薬剤師がすすめやすい低刺激・和ハーブケア」 |

### 3.3 初期ターゲット優先順位

```txt
S：高級旅館
理由：既存実績（強羅花壇）とアメニティ後購入導線が強い。インバウンド/土産/高単価宿泊体験と相性が良い。

A：エステ、美容皮膚科・クリニック
理由：敏感肌・肌荒れ・施術後ケア・ホームケア提案と相性が良い。

A：インバウンド向け店舗、空港/百貨店/免税系
理由：抹茶、富士山、日本由来、海外Amazon展開と接続できる。

B：介護・訪問介護/富裕層向け介護
理由：高齢者の乾燥ケア文脈はあるが、施設直販は予算感に難あり。家族/富裕層/訪問導線の方が有望。

B：美容室・ネイル
理由：既存顧客への物販導線はあるが、優先度は旅館/エステ/クリニックより下。
```

### 3.4 初回営業導線

```txt
1. 電話で業態別課題に接続
2. 福美水の置く理由を短く提示
3. 既存実績を1つだけ添える
4. サンプル送付を打診
5. 送付先・使用者・確認日を決める
6. 使用感確認
7. オンライン商談または訪問商談へ接続
```

### 3.5 サンプル送付の条件設計

サンプル送付は有効だが、送って終わりになると失敗する。  
送付前に最低限以下を確認する。

```txt
・誰が試すか
・何目的で試すか（アメニティ/物販/施術後ケア/高齢者ケア/土産）
・現在使っている商品/アメニティは何か
・切替/追加導入の判断者は誰か
・いつ使用感を確認できるか
・オンライン商談/訪問商談に進む条件は何か
```

## 10. キックオフ議事録要約

### 10.1 契約・支援体制

```txt
営業支援ユニット：7,000ポイント
1コール：約1ポイント
7,000件程度の電話アプローチが可能
電話以外にフォーム、手紙、DM、FAX等も利用余地あり
リストはセールスクラウド上で作成可能
最大6万件までダウンロード可能
```

```txt
セールスユニット：232時間
商談対応、マニュアル作成、ロープレ、営業資料制作等を含む
SPでアポ獲得
SUで商談対応
温度感が高い先、クロージングに近い先を福美人側へトスアップ
最終受注・契約は福美人側で対応
```

### 10.2 支援開始スケジュール

```txt
今回：取材会議・キックオフ
次回：戦略会議
戦略会議でターゲットリストと架電台本を提示
その次の週には電話活動開始可能
SUは商談者アサイン、ロープレ、準備が必要なため実運用まで約1ヶ月
```

### 10.3 営業支援全体の目的

単にアポを獲得することではなく、再現性のある営業の仕組みを作ること。

```txt
どのターゲットリストに対して
どの台本で
どのようにアプローチすると
どれくらいの反応が出るのか
```

を活動記録として蓄積し、最終的に福美人側へ納品する。

### 10.4 会社ビジョン・売上目標

```txt
会社思想：美を通じて人々を幸せにしたい

ブランド思想：100年以上続くものにしたい
昨年売上：約1.3億円
今期見込み：約1.5億円
中長期目標：3年で3億円、または5年で5億円
```
