更新日：2026-07-02  
対象OS：`clients/32_keylab.md`  
用途：SP実戦投入用の架電台本。POINTQUIC利用中 / 資料送付済み既存ユーザーに対して、Zoom個別説明予約を取得する。  
レビュー反映：Claudeレビュー結果を反映。旧案68点 → 実戦投入ラインへ圧縮。

## 0. 台本の目的

POINTQUICをご利用中、またはPOINTQUIC2.0に関する資料をご確認いただいたお客様に対し、現在のPOINTQUIC運用をより効果的に活用できるかを確認するZoom個別説明予約を取得する。

単なる新機能の案内ではなく、以下を確認する場として案内する。

```txt
・今のポイント運用を来店促進につなげられるか
・会員証やポイント機能をより効果的に使えるか
・LINE連携やWeb会員証を今の運用に活かせるか
・現在の設定のままで使えていない機能がないか
・今後の運用に合わせて拡張できる機能があるか
```

最終着地は、予約フォームから日程を選択いただき、鍵野代表とのZoom面談につなげること。

---

## 1. 台本v2で反映した修正方針

Claudeレビューでの主な指摘を採用し、以下の方針へ修正した。

```txt
1. 「資料送付済み」前提の冒頭をやめる
   → 利用中ユーザー / 資料送付済みユーザーの両方に対応できる表現へ分岐

2. 冒頭を短くする
   → 5項目の機能列挙をやめ、1〜2分で関心を取る構成へ圧縮

3. 鍵野代表Zoomの価値を明確にする
   → 「設定や運用の勘所を一番よく知っている鍵野代表が直接確認する」と伝える

4. フォーム送付で終わらせない
   → その場でフォーム入力 / 今週中の仮押さえまで促す

5. 「無理に新機能を入れるわけではない」等の弁明表現を削除
   → 営業っぽさ・GPTっぽさを減らす
```

## 4. 関心確認

```txt
ありがとうございます。

今回のZoomでは、初期設定や新機能の説明だけでなく、
今のPOINTQUICをどう使うとより効果的かを、店舗様の状況に合わせて確認できます。

ちなみに現在、POINTQUICはどのような形でご利用されていますか。
```

### 相手の回答後に確認すること

```txt
ありがとうございます。
今回確認するなら、近いのはどちらでしょうか。

1. ポイント運用や会員証を、もっと来店促進につなげたい
2. LINE連携やWeb会員証を、今の運用で使えるか確認したい
3. まずはPOINTQUIC2.0の設定方法だけ確認したい
```

### 相手が迷っている場合

```txt
大丈夫です。
使い方は店舗様ごとに違いますので、Zoomではまず現在の運用を確認して、
今使える機能、設定しておいた方がよい機能、今は不要な機能を整理する形になります。
```

## 6. 切り返し

### 「今は特に困っていない」

```txt
困っていなくても大丈夫です。

すでに使っている方ほど、
設定や見せ方を少し確認するだけで、会員証やポイント運用をより効果的に使えるケースがあります。

確認だけでも大丈夫ですので、
一度Zoom枠を押さえておきませんか。
```

### 「設定方法だけ知りたい」

```txt
それだけでも大丈夫です。

POINTQUIC2.0の設定方法だけ短時間で確認する形でも問題ありません。
あわせて、今の運用で使える機能があるかも確認できます。
```

### 「LINE連携やPOS連携はまだ考えていない」

```txt
今すぐ連携する前提でなくても大丈夫です。

まずは今のポイント運用や会員証の使い方を確認して、
LINE連携やPOS連携は、必要になったときの参考として見ていただく形でも問題ありません。
```

### 「忙しい」

```txt
承知しました。手短にお伝えします。

今のPOINTQUICをより効果的に使える部分がないか、
Zoomで個別に確認できる枠をご用意しています。

フォームだけ先にお送りしてもよろしいでしょうか。
```

### 「メールで送ってください」

```txt
承知しました。予約フォームをお送りします。

念のため、確認したい内容としては、
会員証、ポイント運用、LINE連携、設定方法のどのあたりが近いでしょうか。

そこに合わせて、Zoomでも確認しやすいようにします。
```

### 「担当ではない」

```txt
承知しました。

POINTQUICのポイント管理、会員証運用、LINE配信などをご担当されている方はいらっしゃいますでしょうか。

その方におつなぎいただけると助かります。
```

### 「代表・上司に確認しないと分からない」

```txt
承知しました。

今回のZoomは、契約判断というより、
まず今の使い方を確認する場です。

日程だけ先に押さえていただいて、
必要であれば上司の方にも共有いただく形で大丈夫です。
```

### 「今回は不要」

```txt
承知しました。

ちなみに、もう設定は完了している感じでしょうか。
それとも、今はあまりPOINTQUICを使っていない感じでしょうか。

状況だけ確認させていただき、必要なタイミングで改めてご案内できればと思います。
```
