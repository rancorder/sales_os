## 2. 商材の核 / USP
- コンセプト：**「会えば伝わる魅力を、会う前から伝える」**。選ぶ前の不安と、選んだ後の不安を下げる映像導線。
- HITO Filmを単なる「映像制作会社」として説明しない。動画納品ではなく、来店前・応募前・問い合わせ前に相手の不安を下げる導線設計として扱う。
- 営業上の再定義：**採用・集客・広報の導線上で、相手が選ぶ前/選んだ後に抱える不安を下げる映像導線設計会社**。
- 思想：動画を1本作って終わりでなく、1回の撮影素材を採用ページ / 店舗紹介 / SNS / YouTube / Meta広告 / TVer広告 / 営業資料 / 展示会資料 / 広報コンテンツへ複数展開する。
- 重要な問いは「どの媒体で流すか」ではなく、**誰の・どの不安を下げるために・どこで見せるか**。
- 効果測定は制作後に考えるのではなく、制作前に「この映像で見る指標」を決める。採用なら応募前理解度・面接前辞退・採用ページ滞在時間、店舗なら視聴完了率・店舗/予約ページ遷移、広報なら滞在時間・問い合わせ・素材再利用回数。
- 提案は最初から大きく始めない。1テーマ / 1導線 / 1拠点 / 1職種などに絞り、小さく試して横展開する。
- 公式サイト上は「沖縄の映像制作会社だし。広告代理店。」という見え方があるが、営業ではこの表現をそのまま押し出すより、採用/集客/広報の導線課題に接続して説明する。

### 2.1 使える言い換え
- 採用：求人票や写真だけでは伝わらない職場の温度を、応募前の候補者に届ける。
- 店舗集客：来店前の不安（雰囲気・スタッフ・サービスの流れ）を先に下げる。
- 多店舗：本部の想いと店舗ごとの魅力を映像で揃え、店舗別の差分も伝える。
- 広報/IR：会社資料や数字では伝わりにくい現場・人・思想を補完する。
- 展示会/営業資料：初回商談前に会社の空気とやっていることを伝え、会社紹介に使う時間を減らす。
- 予約後/応募後/問い合わせ後：申し込み後に再発する「本当にここで良かったか」という不安を下げる。

## 3. 商談設計
- 初回商談の目的は売り込みでなく「貴社の採用・集客・広報のどこに映像を置けば不安が下がるか」を一緒に整理すること。
- 冒頭で「動画を作りませんか」から入らない。相手の現状導線、伝わっていない魅力、選ぶ前/選んだ後の不安を確認してから映像活用に接続する。
- 初回商談のゴールは、受注ではなく**次回、設計案を提示する合意**。
- ターゲット別の置きどころ：
  - 採用：応募前の職場見学。応募前の理解度・志望度を高め、面接前辞退や入社後ギャップを減らす。
  - 多店舗・集客：来店前の接客。店内の雰囲気・スタッフ・サービスの流れ・価格以外の選ばれる理由を伝える。
  - 広報 / IR：数字の背景を見せる。現場・技術・人・思想を補完し、採用/広報/IRを横断する会社理解素材にする。
  - 展示会 / 営業資料：初回商談前に会社理解を前倒しし、商談を会社紹介ではなく課題整理へ進める。
- 商談冒頭：
  - 今日は映像の提案というより、御社の採用・集客・広報の導線上で、どこに「伝わりきっていない不安」があるかを一緒に整理させてください。その結果、映像が必要ないという結論になっても問題ありません。
- 商談中キーフレーズ：
  - 御社の魅力が弱いのではなく、魅力が伝わるタイミングが遅いのだと思います。
  - 来店後、面接後、商談後に伝わる魅力を、来店前・応募前・問い合わせ前に前倒しするのが映像活用の役割です。
  - 映像は、選ばれるためだけではなく、選んだ人に「ここで良かった」と思ってもらうためにも活用できます。
  - 動画を作ることが目的ではなく、どの不安を下げるために、どこで見せるかが大切です。
- NG表現：
  - 「動画を作りませんか」→ 単なる制作営業に見える。言い換え：映像をどの導線で活用できるか整理します。
  - 「売上が上がります」→ 成果保証に聞こえる。言い換え：来店前の不安を下げる導線として活用できます。
  - 「応募が増えます」→ 数だけの話になる。言い換え：応募前の理解度や志望度を高める目的です。
  - 「TVerで成果が出ます」→ 媒体依存に見える。言い換え：目的に応じてTVer等の媒体も選択肢になります。
  - 「大手実績多数です」「HITO Film単体で〇〇社実績」→ 実績の帰属が未確認のため禁止。グループ実績とHITO Film単体実績は分ける。

## 5. 次の一手
1. **対象者拒否率96%の要因分解**
   - リストの問題：採用・店舗・広報/IRのどの課題を持つ企業群かが粗い可能性。
   - トークの問題：「映像制作」「動画制作」の営業に聞こえた瞬間に比較・拒否されやすい可能性。
   - タイミングの問題：採用期・繁忙期・広告検討期とズレている可能性。
2. **電話冒頭は“動画制作”ではなく“伝わる前に離脱している導線”から入る**
   - 例：採用なら「応募前・面接前に職場の雰囲気が伝わりきらず、辞退やミスマッチが起きる場面はないか」。
   - 店舗なら「来店前に雰囲気やスタッフ感が伝わらず、価格比較だけで判断される場面はないか」。
   - 広報なら「資料や数字では現場・人・思想が伝わりきらない場面はないか」。
3. **採用 / 多店舗 / 広報IRでABテストを分ける**
   - 1つのトークで全リストに当てない。
   - リスト属性ごとに「応募前の職場見学」「来店前の接客」「数字の背景を見せる」のどれを入口にするかを分ける。
4. **初回商談では“映像の提案”より“導線診断”に寄せる**
   - 次回提案につなげる確認項目：優先テーマ、伝わっていない魅力、選ぶ前の不安、選んだ後に見返す情報、小さく試すテーマ。
5. **リストはS/A/B/C判定で優先順位を付ける**
   - S：多店舗現場型サービス業、採用難の地方製造業・技術職企業。
   - A：無形サービス、展示会・営業資料に課題があるBtoBメーカー/商社。
   - B：広報/IR、予約後・応募後フォロー課題。
   - C：単発記念動画、価格比較中心、映像導線ではなく素材制作だけのニーズ。
6. **ログ粒度を変える**
   - アポ数だけでなく「採用・集客・広報のどの不安ワードに反応したか」を残す。
   - 「動画は間に合っている」「制作会社がある」「予算がない」の拒否理由を別々に記録する。

### 9.2 リスト判定基準
| 判定 | 条件 |
|---|---|
| S | 多店舗/多拠点、採用難、来店前不安、価格比較、商談前理解不足のいずれかが明確。Web/採用/SNSが古い・静止画中心・店舗別差分が薄い。担当者/決裁者に接続しやすく、横展開可能。 |
| A | 課題は明確だが単店舗寄り、または意思決定に時間がかかる。採用/集客/広報の一領域に映像の置きどころがある。 |
| B | 課題はありそうだが優先度・予算時期が弱い。IR/広報など単価は高いが対象企業数が少ない。追加ヒアリングが必要。 |
| C | 単発記念動画、既存制作会社との価格比較、素材制作だけのニーズ、決裁権が本社/海外で現場前進しにくい。 |

### 9.4 明日から変えるアクション
1. 架電の名乗りから「映像制作会社」を前面に出さない。
2. 受付突破は「採用/店舗集客/広報の見直し」から入る。
3. 「動画を作りませんか」「応募が増えます」系の断定表現を禁止する。
4. 多店舗現場型サービス業と採用難の地方製造業をSターゲットにする。
5. 初回商談のゴールを「受注」ではなく「次回設計案提示」に置く。
6. 制作会社が既にある先には「制作ではなく配置設計」と切り返す。
7. 既存映像の有無ではなく「どこに置いて、誰の不安を下げているか」を確認する。
8. グループ実績とHITO Film単体実績を必ず分ける。
9. リスト判定にWeb/採用/SNSの古さ・静止画偏重・店舗差分不足を見る。
10. SCログではアポ数だけでなく「どの不安ワードに反応したか」を残す。
