## 3. 商談設計（台本改修の核＝最重要）
- 旧台本の「無料/魔法/自動/1円もかからない」訴求は期待値ズレを生み**本人拒否率87-91%**。
- 改修方針：**「当てにいく採用→外しても直せる採用」**。応募増でなく「再現性」で語る。「AI」を前面に出さない（"人が主役・AIはアシスタント"）。
- 台本フロー：現状を「常時/必要時/採用なし」で切り分け→課題特定（応募数/コスト/スピード/再現性）→ズレ言語化（原稿/媒体/ターゲット）→二次商談化。**先に商材価値を語らない**。
- 母集団形成に特化（面接・見極めは顧客の選考ノウハウと組合せ）と役割を切り分け。
- 禁止表現：「無料で採用できます」「応募が集まります」「魔法の求人票」。

## 4. 進捗ログ

### 分析済み｜架電｜2026-05-14〜2026-06-09
**数値**：電話496 / 着電36.9% / 受付拒否25% / 対象者通話11.9% / 対象者拒否91.53% / アポ1（0.2%・むらさき）
**ボトルネック**：本人接続後（対象者拒否91.5%＝接続後ほぼ全滅）
**本人拒否の分類（54件）**：ニーズなし24 / あいまい断り15 / 導入済・他社6 / その他（価格不明含む）4 / タイミング2 / 決裁権なし2 / 自社対応1
**真因4つ**：①AI不信（冷たい/手抜き）②「AI開発」が大きすぎ自分ごと化できず③営業アレルギー（あいまい断り28%）④価格が見えない不安
**所見**：受付・対象者通話は取れるが本人接続後の30秒で崩落。商材説明先行が原因。
→ 架電台本原材料：「AI」を主語にしない／「dodaの応募率を上げる原稿ツール」へ言い換え／冒頭で目安金額を握る／"人が主役・AIはアシスタント"

### 分析済み｜商談｜2026-06-17｜むらさき
**結果**：商談化（既存媒体会社との差別化が論点＝**勝ち候補／二次トスアップ前**）
**刺さった一言**：有料・無料をまたいだ「どの職種・条件・原稿で応募単価が変わるか」の整理。
**詰まった瞬間**：「求人原稿をAIで作れる」だけだと既存媒体会社との違いが不明。
**反論と切り返し**：既存媒体会社の改善は"その媒体内"／リーパは媒体横断で社内にノウハウが残る、と再定義。
→ 商談台本原材料：媒体横断の採用改善ノウハウ蓄積を価値として提示。母集団形成に役割を限定。
