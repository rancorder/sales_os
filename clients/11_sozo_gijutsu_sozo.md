## 2. 商材の核 / USP

### 2.1 技術の一言定義

人工設計ペプチドは、目的の機能を持つ小さな分子を、アミノ酸配列から逆算して設計する技術である。

一般的な素材開発は、天然素材や既存成分を探し、候補を試す流れになりやすい。想像技術創造の強みは、欲しい働きから分子を逆算して設計する点にある。

### 2.2 営業上の言い換え

```txt
× 新しいペプチド素材を使いませんか
○ 御社の研究テーマに合わせて、目的の働きを持つ小さな分子を設計できるか確認したい

× 効果が出ます
○ 作用機序の見立てを立て、可能性を小さく検証できます

× 既存素材より優れています
○ 既存素材だけでは差別化しづらい領域に、設計型の選択肢を追加できます
```

### 2.3 設計できる働き

| 働き | 営業上の説明 |
|---|---|
| 分解する | 狙った物質を切る・分解する働き |
| 結合する | 目的の相手にくっつく働き |
| 膜を通る | 細胞膜などを通り抜ける働き |
| 運ぶ | 届けたい場所へ運ぶ働き |

### 2.4 価値

- 探索・試作回数の削減可能性
- 独自配列による差別化・知財化
- 化粧品、健康食品、医療、診断、機能性材料への応用可能性
- 代表の専門知見を二次商談で活かしやすい

## 3. 商談設計（一次商談の鉄則）

### 3.1 基本思想

人工設計ペプチドは、素材を一方的に売る商談ではなく、先方の研究開発テーマ・製剤課題・顧客接点の中に、目的機能から逆算した分子設計を差し込む商談として扱う。

一次商談では技術を説明しきらない。説明3割、ヒアリング7割。目的は、飯田代表が二次商談で作用機序の見立てを出せるテーマを回収すること。

### 3.2 一次商談の勝ち条件

一次商談は、二次商談化だけが成果ではない。以下のどれかが取れれば前進と判断する。

| 成果 | 内容 |
|---|---|
| A：二次商談化 | 代表・技術者から詳しく聞きたい、特定テーマに当てはまりそうという反応 |
| B：研究テーマ取得 | 腸内環境、抗疲労、ロコモ、オーラルケア、美容、診断など具体領域が出る |
| C：製剤課題取得 | 分解、苦味、吸湿性、安定性、送達性、保持性などの課題が出る |
| D：OEM/顧客ニーズ取得 | OEM先や顧客から独自素材・差別化成分・科学的説明を求められている情報が出る |
| E：紹介先取得 | 製薬、素材、美容医療、研究機関、医師、医療機器メーカーなどの紹介候補が出る |

### 3.3 20分商談の標準流れ

| 時間 | 場面 | 目的 |
|---|---|---|
| 0:00-2:00 | 導入・前提合わせ | 売り込みではなく接点探索であると明示する |
| 2:00-5:00 | 企業理解の確認 | 先方の技術・事業への理解を示す |
| 5:00-8:00 | 想像技術創造の超短縮紹介 | 難しい技術説明を避け、一言で伝える |
| 8:00-15:00 | ヒアリング本番 | 研究テーマ・製剤課題・顧客ニーズを取る |
| 15:00-18:00 | 接点仮説の提示 | 先方発言に合わせて1〜2個だけ仮説を返す |
| 18:00-20:00 | 次アクション合意 | 二次商談、テーマ整理、紹介、資料送付のいずれかへ着地させる |

### 3.4 想定QAの肝

技術詰問（設計手法・MD・精度・安全性）は断定せず、「飯田から正確に説明する」に逃がす。費用は聞かれてから巻末で扱う。安全性・薬機法は絶対に「大丈夫」と言わない。

### 4.2 備前化成株式会社 商談ストーリー

#### 位置付け

備前化成様は、人工設計ペプチドの直接購入先として押すのではなく、B-ReC®／B-MoG®などの「届ける・留める」製剤技術を持つ共創パートナーとして扱う。

```txt
備前化成 = 技術を足す相手
```

備前化成様は、健康食品原料、医薬品原薬、医薬部外品、化粧品、OEM/ODM、製剤技術に強みを持つ。人工設計ペプチド単体の提案ではなく、備前化成様の製剤技術・OEM提案力・既存素材開発に、目的機能から逆算した分子設計を組み合わせることが勝ち筋になる。

#### 勝ち筋

```txt
御社のB-ReC®／B-MoG®の価値を高める“中身の機能分子”を、目的機能から一緒に設計できるか確認したいです。
```

備前化成様が持つ「届ける・留める・高濃度化する」技術は、ペプチドの弱点である分解・送達課題を補完できる可能性がある。一次商談では、研究開発テーマ、製剤課題、OEM先からの要望、既存素材の差別化課題を聞き出す。

#### 確認すべき仮説

| 優先 | テーマ | 確認質問 |
|---|---|---|
| 1 | B-ReC® × 腸内環境 | 腸内環境やマイクロバイオーム関連で、中身の素材側に新規性を持たせたいテーマはあるか |
| 2 | OEM提案力強化 | OEM先から、他社と被らない原料や科学的に説明しやすい素材を求められることは増えているか |
| 3 | B-MoG® × 口腔／美容／免疫 | B-MoG®をオーラルケア以外に、美容・粘膜・免疫系へ展開する構想はあるか |
| 4 | 抗疲労・ロコモ | 抗疲労、筋肉維持、シニア向け機能性食品で、既存素材だけでは差別化しづらい領域はあるか |

#### NG表現

| NG表現 | 理由 |
|---|---|
| 食品素材としてすぐ使えます | エビデンス・制度・製剤検証前の断定になる |
| 効果が出ます | 薬機法・景表法・研究開発商談上のリスクがある |
| 天然素材より優れています | 先方の既存素材事業と対立する |
| ペプチドを買ってください | 共創可能性を潰し、単なる売り込みになる |

### 4.3 創作メディカル株式会社 商談ストーリー

#### 位置付け

創作メディカル様は、人工設計ペプチドの直接導入先ではなく、医療機器・製剤・美容医療・製薬周辺に接点を持つ紹介／共創探索パートナーとして扱う。

```txt
創作メディカル = 道を開く相手
```

架電段階では、先方から「自社が直接ペプチドを扱うことはない」という反応が出ている。そのため、一次商談で無理に導入提案へ進めるとズレる。

勝ち筋は、御社自身への導入ではなく、御社のお客様や開発相談の中で、分子機能設計が必要になる場面があるかを確認すること。

#### 勝ち筋

```txt
御社の医療機器開発・設計相談に、分子機能設計という新しい引き出しを追加できるか確認したいです。
```

創作メディカル様は、医療機器の設計・開発・販売、製剤周辺、注射器・バイアル・美容医療などの接点を持っている。ペプチドそのものよりも、薬剤・有効成分・製剤・デリバリー・処置後ケアなどの周辺相談が入り口になる可能性がある。

#### 確認すべき仮説

| 優先 | テーマ | 確認質問 |
|---|---|---|
| 1 | 紹介先探索 | この話を聞くべきなのは、製薬・素材・美容医療・研究機関のどのタイプか |
| 2 | 医療機器 × 製剤周辺 | 注射・バイアル・自己注射・調製容器の周辺で、薬剤や有効成分側まで絡む相談はあるか |
| 3 | 美容医療 × 皮膚／脂質／術後ケア | 美容外科・形成外科領域で、成分・素材・処置後ケアに関する相談はあるか |
| 4 | 製薬・素材企業への接点 | 独自素材・新規作用機序・差別化成分に関心がありそうな企業はあるか |
| 5 | 開発相談の詰まり | 医療機器の形は見えているが、有効成分や素材側が決まらず止まる相談はあるか |

#### NG表現

| NG表現 | 理由 |
|---|---|
| 御社でペプチドを扱いませんか | 架電段階の「直接扱わない」という前提とズレる |
| 脂肪を減らす、アンチエイジング、治療効果 | 効能断定・薬機法リスクが高い |
| 共同開発しましょう | 現時点では直接ニーズが弱く、重く聞こえる |
| 紹介してくださいと急ぐ | 営業色が強くなり、代表の協力姿勢を損なう |

## 8. 横展開パターン

### 8.1 研究開発型商材の商談OS

研究開発型商材は、初回から導入メリット・価格・事例で押すと浅くなる。重要なのは、先方の未確定テーマを聞き出し、二次商談で専門家が仮説を返せる状態を作ること。

```txt
一次商談 = 未確定テーマの採掘
二次商談 = 専門家による作用機序・PoC仮説の提示
```

### 8.2 直接導入先と探索ハブの分離

同じアポでも、すべてを導入候補として扱わない。

| 類型 | 意味 | 例 |
|---|---|---|
| 直接導入先 | 自社で研究・開発・採用する可能性がある | 研究開発部門、素材メーカー、製剤技術企業 |
| 共創候補 | 自社技術と組み合わせることで価値が出る | 備前化成 |
| 探索ハブ | 自社導入は薄いが、周辺ネットワークや紹介価値がある | 創作メディカル |

### 8.3 営業トークの変換ルール

```txt
売る → 接点を探す
説明する → 仮説を作る材料を取る
導入させる → 次回専門家が見立てを返せる状態にする
失注判定 → 直接導入・共創・紹介・情報交換のどれかに分類する
```

この型は、人工設計ペプチド以外の研究開発型商材、素材、製剤、医療、バイオ、化学、技術顧問型商談にも横展開可能。
