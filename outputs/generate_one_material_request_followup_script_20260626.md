# ジェネレートワン｜本人資料請求フォロー架電台本（実ログ反映版）

## 0. 台本の目的

この台本は、本人資料請求者に対して「資料をご覧いただけましたか？」で終わらせないためのフォロー架電台本である。

目的は、資料の感想確認ではなく、相手の関心軸を確認し、次回の「御社向け企画骨子案」の場に接続すること。

```txt
本人資料請求
↓
資料送付後フォロー
↓
関心軸確認
↓
活用可能性の仮説提示
↓
御社向け企画骨子案の二次商談化
```

## 1. 冒頭トーク

```txt
お世話になっております。
先日、日本企業超会議の資料をお送りした件でご連絡しました、
ジェネレートワンの〇〇です。

資料を細かくご説明したいというより、
御社の場合、採用向け・広報向け・営業前の信頼形成向けの
どこで見るのが近そうかだけ確認できればと思いまして。

今、1〜2分だけよろしいでしょうか。
```

### 架電者メモ

- 「資料見ましたか？」から入らない
- 「どの用途で見るのが近そうか」を確認する
- 相手が資料を見ていなくても責めない

## 2. 資料を見ていない場合

```txt
全然大丈夫です。
今回の資料は一般的な内容なので、
むしろ御社で見るならどの観点が近いかを先に整理した方が、
社内共有もしやすいと思っています。

採用・広報/PR・営業前の信頼形成の中だと、
どれが一番近そうでしょうか。
```

## 3. 資料を見ている場合

```txt
ありがとうございます。
資料の内容で、御社に近そうだなと思われた部分、
逆に少し違うなと思われた部分はありますでしょうか。

もし判断しづらければ、
採用・広報・営業前の信頼形成の3つに分けると、
どこが一番近そうかだけでも大丈夫です。
```

## 4. 関心軸の切り分け質問

```txt
今回の日本企業超会議は、動画を作るだけというより、
企業のリーダーや事業の魅力を番組として見せて、
その後も採用・営業・広報で使えるコンテンツ資産にするものです。

御社で考えると、近いのはどれでしょうか。

1. 採用で、会社の雰囲気や代表・責任者の考え方を伝えたい
2. 広報/PRで、会社や事業の見え方を強くしたい
3. 営業前の信頼形成で、商談前に会社理解を進めたい
4. 経営者や事業責任者を前に出して、信頼感を作りたい
5. 今は具体的ではなく、まず情報収集に近い
```

相手が答えにくそうな場合：

```txt
どれか1つに決めなくて大丈夫です。
強いて言うと、採用・広報・営業の中だと、
どれが一番近そうでしょうか。
```

## 5. 分岐A：採用ブランディングに関心

```txt
採用寄りですね。ありがとうございます。

この企画は、求人広告のように条件だけを見せるというより、
代表や事業責任者の考え方、会社が何を大事にしているかを
番組として見せられるのが特徴です。

候補者からすると、入社前に会社の温度感やリーダーの考え方が分かるので、
応募の質や志望度を上げる材料として使いやすいです。
```

確認質問：

```txt
今、採用でいうと、
応募数を増やしたいのか、応募者の質を上げたいのか、
あるいは入社後のミスマッチを減らしたいのか、
どこが一番近いですか。
```

二次商談への橋渡し：

```txt
もし採用向けで考えるなら、
御社の場合はどんな切り口で見せると候補者に刺さるか、
簡単な企画骨子にしてお話しできます。
```

## 6. 分岐B：広報/PR・会社の見せ方に関心

```txt
広報・会社の見せ方の観点ですね。

ここは日本企業超会議と相性がいいです。
企業のリーダーや事業責任者を、単なるインタビューではなく、
物語の主人公として見せる設計ができます。

なお、出演は必ず社長でないといけないわけではありません。
責任者の方や、会社の顔として語れる方でも成立します。
```

確認質問：

```txt
御社で前に出すとしたら、
代表の方、事業責任者の方、採用や広報の責任者の方だと、
どなたが一番イメージに近いですか。
```

二次商談への橋渡し：

```txt
その方を前面に出すなら、
どんなテーマで見せると御社らしさが伝わるか、
次回は企画骨子案でお話しできます。
```

## 7. 分岐C：営業前の信頼形成に関心

```txt
営業前の信頼形成寄りですね。

この場合、単に動画を作るというより、
第三者の番組に出る形で、御社の事業や考え方を分かりやすく見せるのがポイントです。

営業資料だけだと伝わりにくい会社の信頼感や、
なぜこの事業をやっているのかを、商談前に見てもらえる資産にできます。
```

確認質問：

```txt
営業面で使うとしたら、
初回商談前の信頼づくり、既存顧客への認知、
展示会や紹介時の説明補助、
このあたりだとどれが近そうですか。
```

二次商談への橋渡し：

```txt
御社の営業先に見せる前提で、
どんな番組切り口にすると伝わりやすいか、
一度たたき台をお持ちできます。
```

## 8. 分岐D：予算・時期が理由で保留

```txt
承知しました。
今期すぐに実施する前提ではなくても大丈夫です。

むしろ来期施策や年度末の予算検討に向けて、
御社の場合どんな切り口の番組企画にできるかだけ
先に見ておくと判断しやすいと思っています。

一般説明ではなく、御社向けの簡単な企画骨子案として
15〜20分ほどお話しする形であれば、確認の場をいただけそうでしょうか。
```

## 9. 分岐E：社内共有・担当部署確認型

```txt
ありがとうございます。
では資料を社内でご確認いただく前提で、
どの部署に共有いただくのが近そうかだけ確認させてください。

採用広報、コーポレート広報、営業企画、経営企画の中だと、
どちらが一番関係しそうでしょうか。
```

本人を紹介者に変える一言：

```txt
もし可能でしたら、資料をお渡しいただく際に、
「採用・広報・営業活用のどれで見るかを確認してほしい」と
一言添えていただけると、判断しやすいと思います。

社内共有後、来週あたりにご感触だけ確認してもよろしいでしょうか。
```

## 10. YouTube広告と誤認された場合

```txt
ありがとうございます。
一点だけ補足すると、YouTube広告枠のご提案ではありません。

動画の間に広告を流すというより、
企業の代表者様や事業責任者様に番組へ出ていただき、
その内容を採用・営業・広報で使えるコンテンツ資産にする企画です。

なので広告出稿というより、
御社の魅力や考え方を第三者メディア上で伝える施策に近いです。
```

## 11. 動画制作と誤認された場合

```txt
ありがとうございます。
動画制作会社として撮影だけを請けるというより、
番組の企画、出演設計、見せ方、撮影後の活用まで含めた企画枠です。

比較対象としては、単なる動画制作よりも、
採用広報・営業資料・企業ブランディングの資産づくりに近いです。
```

## 12. 二次商談への打診

```txt
もし方向性が少しでも合いそうであれば、
次回は一般的なサービス説明ではなく、
御社の場合どんな企画にできそうか、
簡単な骨子案ベースでお話しできればと思っています。

来週あたり15〜20分ほど、
採用・広報・営業活用のどこを軸にするかだけ
一度すり合わせできませんでしょうか。
```

日程打診：

```txt
来週ですと、〇曜日の午前か、〇曜日の午後でしたら、
どちらがご都合よろしいでしょうか。
```

同席促進：

```txt
もし採用や広報、営業企画のご担当者様も関わるようでしたら、
その方もご一緒の方が話が早いかもしれません。
```

## 13. 断り代替っぽい場合

```txt
承知しました。
不要であればもちろん大丈夫です。

最後に一点だけ、
御社でこういった番組出演型のコンテンツを使うとしたら、
採用・広報・営業の中だとどれが一番近そうですか。

そこだけ分かれば、今後ご案内する場合も不要な内容は避けられます。
```

回答がある場合：

```txt
ありがとうございます。
では、その観点に絞って資料を見ると判断しやすいと思います。
〇日頃に一度だけ、その後のご感触を伺ってもよろしいですか。
```

回答がない場合：

```txt
承知しました。
では、今回はいったん資料だけご確認いただければ大丈夫です。
不要でしたらもちろん結構ですので、また時期が合いそうな際にご案内します。
```

## 14. 共通切り返し

| 相手の反応 | 返し方 |
|---|---|
| まだ資料を見ていない | 「大丈夫です。資料の細かい説明ではなく、御社ならどの用途が近いかだけ確認できればと思っています」 |
| 興味があれば連絡する | 「ありがとうございます。では、こちらからも〇日頃に一度だけ、その後のご状況を伺ってもよろしいですか」 |
| 担当者に確認する | 「ありがとうございます。では、ご担当者様が見る時に判断しやすいよう、採用/広報/営業のどこで見るべきかだけ整理してお送りします」 |
| 予算がない | 「承知しました。今すぐ決める話ではなく、予算化前の判断材料として、御社ならどう使えるかだけ見ていただく場でも大丈夫です」 |
| YouTube広告はやっていない | 「広告枠ではなく、番組出演を起点にしたコンテンツ資産化の話に近いです」 |
| 動画制作は間に合っている | 「動画制作というより、番組出演をきっかけに採用・営業・広報で使える資産を作る話に近いです」 |
| 社長出演は難しい | 「社長でなくても大丈夫です。責任者の方や、会社の顔として語れる方でも成立します」 |
| 今は必要ない | 「承知しました。では時期が合う前提で、〇月頃に一度だけ情報提供してもよろしいでしょうか」 |

## 15. NGトーク

- 「資料をご覧いただけましたか？」だけで終わる
- 見ていない相手に、資料の内容を最初から説明し直す
- 相手の関心軸を聞かずに、商材説明を続ける
- 「YouTube広告です」と受け取られたまま進める
- 「動画制作です」と表現して、制作会社の相見積もり枠に入る
- 社長出演が必須のように話す
- 「興味があればご連絡ください」で終える
- 社内共有先・判断部署・次回確認時期を聞かない
- 断られた後に粘りすぎて、次回連絡の許可まで失う

## 16. 架電者向け注意点

- 本人資料請求者は、新規架電より温度が高いが、高温リードとは限らない
- フォローの主語は「資料」ではなく「御社でどう使えそうか」
- 相手が資料を見ていなくても、責めない
- 採用・広報・営業・予算のどれに近いかを必ず記録する
- YouTube広告・動画制作の誤認は即座に修正する
- 二次商談の勝ち筋は「一般説明」ではなく「御社向け企画骨子案」
- 個人名・電話番号・メールアドレスはGitHubに残さない。ログ原本はDriveを参照する
