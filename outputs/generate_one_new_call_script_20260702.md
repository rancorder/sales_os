## 1. 今回の判断

新規架電台本は、既存の「資料請求フォロー台本」と同じものを使い回さない。

```txt
新規架電
= 完全初回接点。受付突破・担当部署特定・関心軸取得・再連絡許可が勝ち。

資料請求フォロー
= 本人接触済み。資料送付後に用途・社内共有先・企画骨子案の場取りへ進める。
```

営業導線は以下で運用する。

```txt
新規架電
↓
用途切り分け
↓
資料送付 or 企画骨子案打診
↓
資料請求フォロー台本へ接続
↓
二次商談：御社向け企画骨子案
```

## 2. 新規架電台本の勝ち筋

- 初回電話では商材を長く話さない。
- 「日本企業超会議」という企業紹介番組として入り、採用広報・広報PR・営業前の会社紹介のどこで使えそうかを確認する。
- 「YouTube広告」「動画制作」「広告出稿」と最初から言わない。
- 社長出演を必須にしない。代表者様・事業責任者様・会社の顔になる方、という言い方にする。
- 資料送付だけで終えず、次回連絡日か企画骨子案の場を取る。

```txt

4. 経営者や事業責任者を前に出して、会社の考え方を伝えたい
5. 来期施策や予算検討の判断材料として見たい
```

## 6. GPT臭い表現の禁止

架電トークでは、以下を原則使わない。

- 「〜というより」
- 「単に〜ではなく」
- 「〇〇ではなく、△△です」
- 「資産化」
- 「第三者メディア上で」
- 「信頼形成」

電話では概念語より、何に使えるかを先に出す。

## 7. 架電後の記録項目

| 記録項目 | 選択肢・記入例 |
|---|---|
| 受付突破 | 突破 / 不在 / 拒否 / 部署不明 / 代表番号不可 |
| 接続部署 | 採用 / 広報PR / 営業企画 / 経営企画 / 総務 / 不明 |
| 本人接続 | 接続 / 不在 / 受付止まり / 別部署案内 |
| 関心軸 | 採用 / 広報PR / 営業前の会社紹介 / 経営者・事業責任者発信 / 予算検討 / 不明 |
| 温度感 | A：企画骨子案OK / B：資料送付後フォロー / C：担当部署確認 / D：時期フォロー / E：停止 |
| 誤認 | YouTube広告 / 動画制作 / 広告枠 / 社長出演必須 / 費用不明 / なし |
| 次回アクション | 二次商談打診 / 資料送付 / 担当部署確認 / 時期指定フォロー / 深追い停止 |
| 次回連絡日 | ○月○日頃 / 来週前半 / 社内共有後 / 予算検討前 |
| ログメモ | 相手が反応した言葉、拒否された言葉、次回使える仮説 |

## 8. 今後の架電ログ分析プロトコル

今後ログが入ったら、アポ数だけで見ない。

### 8.1 定量で見る項目

- 総架電数
- 受付突破率
- 本人接続率
- 部署特定率
- 関心軸取得率
- 資料送付率
- 再連絡許可率
- 企画骨子案打診率
- 二次商談化率
- 停止率

### 8.2 定性で見る項目

- 受付で止まった言葉
- 本人が反応した用途
- 誤認された表現
- 「資料だけ」になった理由
- 担当部署がずれた理由
- 予算・時期・決裁者が拾えたか
- 二次商談に進んだ時の言い回し
- 深追いすべきでない拒否パターン

### 8.3 ログ分類

| 分類 | 状態 | 次アクション |
|---|---|---|
| A：企画骨子案OK | 用途が明確で15〜20分の場が取れる | 二次商談化 |
| B：資料送付後フォロー | 興味はあるが即日程化しない | 資料請求フォロー台本へ接続 |
| C：担当部署確認 | 本人・受付が部署を探している | 部署確認後に再架電 |
| D：時期フォロー | 予算・来期施策・タイミング待ち | 時期指定フォロー |
| E：停止 | 用途なし・拒否強い・再連絡不可 | 深追い停止 |
