# 福美人株式会社｜Membersチャット重要事項整理

更新日：2026-06-30  
対象：Membersチャットエクスポート要約  
格納方針：個人連絡先、会議URL、メールアドレス、契約原本、請求詳細は記載しない。営業戦略・運用判断に必要な要点のみ残す。

---

## 1. 結論

Membersチャット確認により、福美人案件の主戦略は以下で確定寄り。

```txt
主商材：福美水
呼び出し先：仕入れのご担当者
着地：サンプル送付
主ターゲット：旅館、エステ、美容室、美容関係、クリニック、調剤、介護、訪問介護、卸/パートナー
```

以前の「人事・採用向け研修/福利厚生」仮説は、現運用の主戦略から外す。

---

## 2. OSへ反映すべき重要事項

### 2.1 呼び出し先

トーク作成依頼では、呼び出し先が `仕入れのご担当者` とされている。  
よって、受付突破では人事・採用ではなく、以下の担当者接続を優先する。

```txt
・仕入れ担当
・アメニティ担当
・物販担当
・ホームケア商材担当
・施設/店舗の導入判断者
```

### 2.2 ターゲットリスト

確認できた主なリスト/セグメント。

```txt
2606_旅館運営＠全国
2606_エステ業界、美容室、美容関係＠全国
2606_クリニック関連、調剤＠全国
2606_介護業界、訪問介護＠全国
化粧品販売＠全国 6016件
```

### 2.3 業態別方針

```txt
旅館：売上規模、単価、一休掲載などで質を担保。低価格帯は除外候補。
クリニック：皮膚科/美容皮膚科など、肌悩みと接続しやすい先を優先。
エステ/美容室：ホームケア・物販導線として提案。
介護：施設直販だけでなく、訪問介護・ケアマネージャー周辺も検討。
卸：オンライン商談着地でも進めやすい。
```

---

## 3. 台本修正ポイント

### 3.1 実績訴求

単なる導入実績の羅列ではなく、以下を強調する。

```txt
・15周年
・累計190万本販売
・肌質を選ばず、敏感肌から利用可能
・男性にも使える
```

### 3.2 旅館向けの表現注意

具体的な宿泊施設名を出すとトラブルリスクがあるため、架電台本では抽象表現へ置き換える。

```txt
NG寄り：具体的な宿泊施設名を多用
OK：高級老舗旅館、5つ星ホテル、高級宿泊施設など
```

### 3.3 卸/パートナー向け

一般流通では未着手であることを、弱みではなく市場の広がりとして訴求する。

```txt
他の店舗様ではまだ未着手の領域も多く、先駆けて導入いただける余地があります。
```

### 3.4 キャンペーン

導入後押しとして、ケース購入時のミニボトルプレゼントなどの特典を台本に反映する。

## 4. サンプル送付時の必須ヒアリング

サンプル送付は有効だが、送って終わりにしない。  
送付時に必ず以下を確認する。

```txt
1. 導入希望のアイテム
2. 導入希望時期
3. 導入希望数量
```

加えて、以下も確認できると商談化しやすい。

```txt
・誰が試すか
・何目的で試すか
・現在使っている商品/アメニティは何か
・切替/追加導入の判断者は誰か
・いつ使用感を確認できるか
```

送付物には、商品概要と導入までの流れを記載したA4資料または手紙を同封する。

## 8. べー判断

このチャットで一番重要なのは、主戦略のズレ修正。

```txt
人事・採用向けの福利厚生/研修
ではなく、
福美水を仕入れ担当者へ売る販路開拓案件。
```

勝ち筋はこれ。

```txt
商品説明
↓
ではなく
↓
業態別の導入理由
↓
サンプル送付
↓
使用感確認
↓
導入希望アイテム/時期/数量確認
↓
商談化
```

最重要の修正ポイント。

```txt
1. 呼び出し先を仕入れ担当へ修正
2. 高級旅館の実名訴求は抽象化
3. 15周年・累計190万本・敏感肌から使える実績訴求を追加
4. サンプル送付時に3点ヒアリングを必須化
5. 稼働0件IDの原因確認
```
